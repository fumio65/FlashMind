Admin User Mgmt
1
Role Promotion (FR-07)"Promote" opens a dropdown: Student → Creator → Admin. PUT /api/admin/users/:id/role updates the role field in the User model.
2
Suspend / BanSuspend = temporary (user can be restored). Ban = permanent. Suspended users receive 401 on login with a "suspended" message.
3
Search + FiltersFilter by role or status. Real-time search by name or email. "🚫 Suspended" tab shows problem accounts for quick review.
4
User StatsDecks and sessions columns help identify power users (Creator candidates) and low-activity accounts.
